# 音频播放器

基于python、pysdl2的音频播放器：

### windows 配置
 * python 版本2.7.15
 * 下载PySKL2-0.9.5.zip
 * 下载SDL2-2.0.9-win32-x64.zip
 
#### 安装
 * 安装PySKL2
   + python setup.py install
   + 安装完成
 * 解压SDL2*.zip
   + 解压出SDL2.dll文件。可以放到任意位置。如：PYTHON_HOME/DLLs下
   + 配置环境变量PYSDL2_DLL_PATH=%PYTHON_HOME%/DLLs
   

#### 运行
 * 执行 python sdl2-audio.py
