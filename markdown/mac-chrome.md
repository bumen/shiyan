## Mac chrome快捷键

### 窗口快捷键
 * 新建窗口  
   `command + N`
 * 打开新标签页  
   `command + T`
 * 在后台打开新标签页  
   `command + 点击链接`
 * 在前台打开新标签页  
   `command + shift + 点击链接`
 * 在新窗口中打开链接  
   `shift + 点击链接`
 * 切换标签页  
   `command + option + 左右`
 * 前近与后退  
   `command + 左右`
 

### 网页快捷健
 * 重新载入当前网页  
   `command + R`
 * 停止载入当前网页  
   `command + .`
 * 将选中内容作为查找关键字  
   `command + E`
 * 查找关键字  
   `command + F`
 * 打开开发人员工具  
   `command + option + I`
 * 将当前网页保存为书签  
   `command + D`
 * 放大  
   `command + -+`
 * 直接恢复正常大小  
   `command + 0`
 * 向下滚动页  
   `空格`
 * 在当前搜索网页  
   `command + option + F`
 * 向下滚动  
   `option + 下`
 * 向上滚动  
   `option + 上`
 
   
### 地址栏快捷健
 * 选中网址  
   `command + L`
 
### 功能快捷键
 * 打开和关闭书签栏  
   `command + shift + B`
 * 清除历史记录  
   `command + shift + backspace`
