#### ProviderInfo 带表一个服务端地址
 * 可以看equals方法
   + port相同
   + rpcVersion
   + protocolType
   + host
   + path
   + serializationType
   + 都同，则带表相同服务端地址
   
 * 每个ProviderInfo都跟据协议类型创建一个ClientTransport
 
 * ProviderInfo为负载均衡使用
