#### 方法调用
 * 考-方法调用限制
    ```
        public void run() {
          m(1, 2);  //是否可以编译通过
          //答：不可以。因为1, 2是int类型, 但m方法参数类型是byte, int类型
          //   当方法调用时，不允许数值类型隐试窄化。因为如果有重载方法时，导致判断过程复杂。所以java不允许这么作
          //修：使用强制类型转换
        }
        public int m(byte a, int b) {
            return a + b;
        }
    ```
